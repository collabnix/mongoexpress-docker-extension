## Getting Started


```
 git clone https://github.com/collabnix/mongoexpress-docker-extension
```

## Build the Extension

```
 make build-extension
```

## Install the Extension

```
 docker extension install ajeetraina/mongodb-express-docker-extension:1.0
```
